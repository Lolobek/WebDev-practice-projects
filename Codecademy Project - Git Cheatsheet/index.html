Git Cheatsheet

Main Commands

Słowo Definicja
Repozytorium folder, w którym Git śledzi projekt i historię jego zmian
Clone wykonanie kopii repozytorium z chmury na komputer
Stage informacja dla Gita, które zmiany mają zostać zapisane następne
Commit zapisanie zmian i nadanie im opisu
Branch równoległa linia czasu, tj. praca nad nową wersją bez ingerowania w starą lub równoległa praca kilku osób na kilku gałęziach
Merge połączenie zmian w różnych gałęziach (branches)
Pull ściągnięcie najświeższych zmian w repozytorium w chmurze
Push przekazanie swoich zmian do repozytorium w chmurze
